
# Allergen Alert

Allergen Alert is an extension made to help people with allergies avoid foods that could cause allergic reactions. This is done by warning the user when they open a recipe that contains the selected allergens/food restrictions. Allergen Alert also offers substitutes for some common allergens.

## Files:
- App: The main runner file. Either the GUI or CLI must be uncommented to run.
- Search: The main class, makes and searches through an array, looking for allergen/keywords.
- Substitute: An extension of the Search class, has functionality to give substitutes for detected allergens
- Input: A class that takes CLI user input for changing settings
- GUI: runs the GUI, similar functionality to the CLI
- FileReader: Provides functionality to read through a file and seperate it by newline

## Features
- Interactive GUI or CLI interfaces, depending on preference
- Ablity to search and flag certain keywords/allergens from a preset list
- Ability to give allergen substitute suggestions
